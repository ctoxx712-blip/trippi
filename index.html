📤 傳個行程畀朋友

最簡單：複製一條連結，WhatsApp／Email 傳過去。對方一撳就直接開 Trippi 收到成個行程 — 唔使註冊、唔使設定、咩裝置都得。

其他方式（郵件・分享碼・檔案）
行程較大：分享碼未必放得入郵件，請改用「下載 .json」做附件。
郵件會附上文字行程＋分享碼；如果郵件 App 冇自動彈出，分享碼已幫你複製咗，直接貼落去就得。

🤝 多人協作編輯

一個人撳「建立協作房間」，其他人喺自己部機開 Trippi，掃 QR 或者輸入 6 位代碼就即刻加入 — 任何裝置、任何網址都得，修改即時互相同步（以最後儲存為準）。內容有加密，但知道代碼嘅人就睇到行程，唔好公開貼代碼。

☔ 天氣預報（免費・免 API key）

資料來自 Open-Meteo（免費、免註冊）。只有未來約 16 日內嘅日子先有預報；落雨日會自動喺「旅遊建議」提你改室內景點。

🍜 附近搵嘢（免費・免 API key）

資料嚟自 OpenStreetMap（免費、免註冊）。以你揀嘅景點或 GPS 位置為中心搵；撳「➕」加入當日行程時會自動帶埋地址、營業時間同公休日。

🧭 導航模式（當日）

GPS 需要你授權（手機 Safari 最準）；「🧭」會以你而家嘅位置開 Google Maps 導航；距離為直線估算。遲到可以一鍵順延剩餘行程。

☁️ 跨裝置同步（選填）

呢個係選填嘅 — 淨係想傳行程畀朋友嘅話，用「📤 傳連結」就得，唔使掂呢度。
⚠️ Deno Deploy 而家暫停新用戶註冊（403 SIGNUP_UNAVAILABLE），所以請用 Cloudflare：撳「📋 複製 Worker 程式碼」→ dash.cloudflare.com 註冊（免費、唔使信用卡）→ Workers & Pages → Create → Deploy → Edit code 貼上 → Deploy → Storage & Databases → KV → Create namespace → 返 Worker → Settings → Bindings → KV namespace，Variable name 一定要打 TRIPPI → Save → 將 workers.dev 網址貼返上面 → 測試連線。

✈️ 航班時刻 API（選填）

有 key 嘅話「⚡ 自動填」會填埋真・起降日期時間＋航廈閘口（自行去 aviationstack.com 免費註冊，每月約 100 次；key 只儲存喺你部機）。冇 key 時 Trippi 會用航線庫＋距離自動估算抵達時間。

📲 安裝到 iPhone / Android（似 App 咁用）

iPhone：用 Safari 開呢個檔案或網址 → 撳底部「分享」→「加入主畫面」，桌面就會有 Trippi 圖示，全螢幕好似 App 咁用。

Android：用 Chrome 開 → 右上選單 →「加到主畫面／安裝應用程式」。

想畀朋友都用到：將呢個檔案上載去免費寄存（Netlify／Vercel／Cloudflare Pages），分享網址就得。每個人嘅帳號同行程會儲存喺自己部機。

ℹ️ Trippi

單檔網頁 App：行程、時間軸衝突偵測、多幣值拆帳與最少轉帳結算，全部在你的裝置上運作。

需要網路的功能：地圖顯示、景點定位、即時匯率更新。其餘功能離線可用。

真・Email 驗證登入與雲端多裝置同步屬下一階段（需後端）。
